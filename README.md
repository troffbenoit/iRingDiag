iRingDiag

Overview

iRingDiag is an iOS engineering diagnostic application developed to assist CT field service engineers in identifying and troubleshooting ring artifacts on Toshiba Aquilion CT scanners.

Ring artifacts are typically caused by detector calibration errors, defective detector channels, or related data acquisition hardware issues. The application provides a portable diagnostic tool that allows service engineers to analyze image data directly from an iOS device while working in the field.

Features

* Analyze CT image data for ring artifacts
* Assist in detector diagnostics and troubleshooting
* Help identify detector channel and calibration issues
* Provide engineering support during field service
* Designed specifically for Toshiba Aquilion CT scanner platforms

Intended Use

iRingDiag is intended for use by qualified CT service engineers as a diagnostic and troubleshooting tool. It is designed to improve service efficiency by providing a repeatable method for evaluating ring artifacts and identifying potential detector or acquisition hardware problems.

Development

* Platform: iOS
* Language: Objective-C
* Development Environment: Xcode
* Target Users: CT Field Service Engineers

Building

1. Open the iRingDiag.xcodeproj project in Xcode.
2. Select the desired iOS target or simulator.
3. Build and run the application.
