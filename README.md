iRingDiag

Overview

iRingDiag is an engineering diagnostic utility developed to assist field service engineers in identifying and troubleshooting ring artifacts on Toshiba Aquilion CT scanners.

Ring artifacts are typically caused by detector calibration errors, defective detector channels, or related acquisition hardware issues. This application provides a streamlined method for evaluating image data and helping isolate the source of these artifacts during service and repair.

Features

* Analyze CT image data for ring artifacts
* Assist in detector diagnostics and troubleshooting
* Aid in identifying detector channel and calibration issues
* Support engineering evaluation during field service
* Developed specifically for Toshiba Aquilion CT systems

Intended Use

This application is intended for use by qualified CT service engineers as a diagnostic and troubleshooting tool. It is designed to improve service efficiency by providing a repeatable method for evaluating ring artifacts and identifying potential hardware or calibration problems.

Development

* Language: Objective-C
* Platform: macOS
* Development Environment: Xcode

Building

1. Open iRingDiag.xcodeproj in Xcode.
2. Build the project using a supported version of Xcode.
3. Run the application.
